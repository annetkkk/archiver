# Archiver

Учебный командный проект на Python: консольная утилита для архивации, разархивации и простого шифрования файлов.

Проект был разработан в рамках курса Python. Основной фокус — работа с файловой системой, аргументами командной строки и разными алгоритмами архивации.

---

## Что реализовано

- консольный интерфейс через аргументы командной строки;
- архивация и разархивация файлов/директорий;
- поддержка нескольких форматов/алгоритмов;
- простое шифрование архива по паролю;
- валидация входных данных;
- модульная структура проекта.

Проект выполнялся в команде из 3 человек.

Мой вклад:
- реализация tar.gz;
- участие в разработке архитектуры;
- работа с аргументами командной строки и обработкой файлов.

## Поддерживаемые режимы

- compress — сжатие/архивация;
- decompress — распаковка/разархивация.

---

## Поддерживаемые типы архивов

- zip
- tar
- rle
- huffman

---

## Пример запуска

Архивация:

```bash
python main.py input_directory -m compress -t zip -o output_archive
```

Разархивация:

```python main.py archive.zip -m decompress -t zip -o output_directory
```

Архивация с паролем:
```python main.py input_directory -m compress -t zip -sp True -o output_archive
```
